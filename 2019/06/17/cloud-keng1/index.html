云就是坑

发表于 2019-06-17

微软的Azure不是完美的，但是的确方便！
微软的Azure可以说是微软现阶段个人认为最牛的产品了，在这个风起“云”涌的云时代，现在仅有谷歌云，AWS和阿里（暂且算一个吧）能和他技术上抗衡。下面讲讲怎么用Azure。
微软的account应该都有吧,

这个申请就不说了，如果你有公司账号使用微软的Office365服务也可以申请。按照提示一步步来，加一个信用卡就OK了。
Azure坑爹第一弹：

170欧免费额度

真的很诱人！12个月免费，170欧额度和外加的25个免费服务。第一个月可以很任性的使用！

但是已经要小心，12月仅是针对一些相应的简单服务免费，170欧只针对第一个月。也就是说任性只有一个月。如果你买了个服务30块一个月，第一月170欧cover了。但是第二月开始就要pay-as-you-go了。也就是你自己需要付这30块钱了。

令人痛苦的Protal

总的来说，Azure的界面是让人满意的。但也是有bug，我遇到最痛苦的bug是：
在Azure的France地区建了一个subscription，玩了两天。还是想迁回之前的Europe West，于是删了之前的subscription， 建了个新的。建了之后就发现所有的VM没有办法选择size了。 彻底崩溃，试了好几天，最后放弃了。只好用Azure的cloud shell来解决这个问题。

Resize

事实上自己建立的VM仅仅是个给blog做测试的地方，但是本身也没期望着能成什么气候，就只买了最简单的B1|s业务，一个月4块钱不到。但是等自己跑blog才发现这个配置压根不行。设备天天报错，IOP总是超负荷（也许是DDos，没有验证，等有钱了前面架个防火墙）。于是只能升级！虽简单升级但是价格涨一倍。不过无所谓了， 就当是每个月中午少吃一顿饭了。

配宝马，配奔驰

Azure总有一种收费像配宝马奔驰的感觉，基础款看看25万不到。但是，你要是想跑你的业务，需要加这个配置那个配置，零零总总加起来价格就翻倍了。当然了，没有宝马奔驰这么夸张，但是还是让人心里不舒服。

总结

总的来说Azure是一款非常值得推荐的产品。易用，易上手！可视化做的相当不错，比起传统的搭环境要直观，而且可以很快布放，容易扩展。对于企业的数据中心是非常好的拓展，延伸及替代，值得认真研究。

记录一个人小目标

今年考完CCIE SEC，明年考个Azure Architecture Expert。至于能不能完成吗？听天由命，成事在天。
